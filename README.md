# SaaS Subscription Platform

## Overview

SaaS Subscription Platform is a basic full-stack web development project that allows users to view subscription plans, register, log in, and access a dashboard.

## Features

* Home Page
* User Registration
* User Login
* Dashboard
* Subscription Plans

  * Basic Plan – ₹99/month
  * Standard Plan – ₹199/month
  * Premium Plan – ₹299/month
* Responsive User Interface

## Technologies Used

* HTML
* CSS
* JavaScript
* Node.js
* Express.js

## Installation

1. Clone the repository:

```bash
git clone https://github.com/gannojiakshaya/SaaS-Subscription-Platform.git
```

2. Install dependencies:

```bash
npm install
```

3. Run the application:

```bash
node server.js
```

4. Open the browser and visit:

```text
http://localhost:3000
```
